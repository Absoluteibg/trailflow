# Run and deploy Trailflow

This contains everything you need to run your app locally.

Trailflow is a local-first autonomous coding assistant with Web and Telegram interfaces.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Configure your environment in `.env` (see `.env.example`)
3. Run the app:
   `npm run dev`
